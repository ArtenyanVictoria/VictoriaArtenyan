## Описание

Codewars является одной из основных платформ, которые я использовала для изучения Python и развития алгоритмического мышления.

Решение большого количества задач позволило мне получить практический опыт в следующих темах:

* работа со строками;
* списки, словари и множества;
* функции и рекурсия;
* сортировки и поиск;
* математические алгоритмы;
* обработка данных;
* оптимизация решений.

В процессе решения задач были развиты навыки:

* анализа условий задачи;
* поиска эффективных алгоритмов;
* написания читаемого кода;
* тестирования и отладки программ.
