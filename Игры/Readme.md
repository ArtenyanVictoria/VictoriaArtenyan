# Крестики-нолики

Консольная реализация игры «Крестики-нолики» на Python.

Игрок соревнуется с компьютером. Игровое поле отображается в текстовом виде, а взаимодействие с программой осуществляется через терминал.

Реализовано:

* отображение игрового поля;
* обработка ходов пользователя;
* случайный выбор хода компьютером;
* проверка победителя;
* определение ничьей;
* проверка корректности вводимых данных.

Использованные инструменты:

* Python
* модуль random

Использованные возможности Python:

* функции;
* циклы;
* условные конструкции;
* списки;
* работа с пользовательским вводом.

Полученные навыки:

* проектирование логики игры;
* организация программы из нескольких функций;
* работа со случайными событиями;
* обработка различных игровых сценариев.
