# Регистрация и вход пользователя

Консольное приложение для регистрации и авторизации пользователей.

При регистрации программа:

* проверяет уникальность логина;
* проверяет корректность логина и пароля;
* сохраняет данные пользователя в отдельный файл.

При входе программа:

* проверяет существование пользователя;
* сверяет введённый пароль с сохранённым;
* сообщает об успешной или неуспешной авторизации.

Использованные модули:

* os

Использованные возможности Python:

* работа с файлами;
* циклы и условия;
* строки и методы строк;
* обработка пользовательского ввода.

Полученные навыки:

* организация хранения данных;
* проверка корректности пользовательского ввода;
* работа с файловой системой;
* разработка логики регистрации и авторизации пользователей.
