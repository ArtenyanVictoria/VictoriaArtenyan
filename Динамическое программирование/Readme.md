# Динамическое программирование

В данной папке собраны задачи, в которых изучались методы динамического программирования и оптимизации алгоритмов. Основное внимание уделялось 
хранению промежуточных результатов, сокращению количества вычислений и поиску эффективных способов перебора вариантов.

## Последовательность

Описание:
Вычисление элементов последовательности, заданной рекуррентными соотношениями.

Использованные методы:

* динамическое программирование;
* bottom-up подход;
* хранение промежуточных результатов в массиве.

## Диоген

Описание:
Представление числа в виде суммы не более трёх треугольных чисел.

Использованные методы:

* предварительное вычисление треугольных чисел;
* метод двух указателей (two pointers);
* оптимизация перебора.

## Рыбалка (Кузнечик)

Описание:
Поиск максимального количества рыбы при ограничениях на перемещение между стоянками.

Использованные методы:

* динамическое программирование;
* хранение состояний;
* поиск оптимального перехода между состояниями.
