Задача Последовательность
Последовательность задана следующим образом:
f(1) = 1, f(2) = 2
Если n чётное, то f(n) = f(n-1) · n/2
Если n нечётное, то f(n) = f(n-1) + f(n-2)
Несколько первых членов последовательности:
1, 2, 3, 6, 9, 27, 36, …
Вычислите значения членов последовательности с номерами 19 и 20.
Числа запишите в строку через один пробел.

Задача Диоген
Говорят, греческий философ Диоген жил в бочке. Может быть и жил, а может быть это была и не бочка. Или очень большая бочка. 
В таких бочках в Испании хранят вина. Вообще эти бочки огромные. Каждая с небольшую комнату. Их можно сложить и получить пирамиду. 
А уже из этой пирамиды сделать вполне комфортабельное жилище. Понятно, что пирамида получается только из определенного количества бочек. 
То число, для которого можно получить пирамиду, называется треугольным. Все остальные числа можно представить как сумму треугольных чисел. 
Очевидно, что 1 — треугольное число, и можно любое число разложить на сумму единиц. Однако существует теорема:
Всякое натуральное число — либо треугольное, либо сумма двух или трёх треугольных чисел.
Выдвинул эту теорему французский математик Пьер Ферма в 1638 году, как водится, без доказательства, но обещал представить его в отдельной статье, 
которая так никогда и не появилась. Доказал это утверждение для треугольных чисел немецкий математик Карл Фридрих Гаусс в 1796 году.
Предположим, что имеется n бочек. Из этих бочек необходимо сложить не более трёх зданий, укладывая их в такие треугольные пирамиды.
Написать программу, которая для заданного общего числа имеющихся бочек подбирает необходимое количество бочек для постройки каждого из не более трёх 
зданий-пирамид.

Входные данные:
одно натуральное число в пределах 1 ≤ n ≤ 10⁸

Выходные данные:
не более трёх натуральных чисел, записанных в одну строку через пробел.

Примеры:
Для n = 14 вывод может быть такой:
10 3 1

Для n = 18 вывод может быть такой:
15 3

Задача Вирус-книголюб(2)

В одной из библиотек появился вирус.
Вирус — это последовательность символов (не более 10).
Вирус внедрялся в текст книги (не более 500 символов), искал в тексте этой книги последовательность символов, которая в точности соответствовала 
его собственной. Из этой найденной последовательности вирус составлял свою копию (реплику). Вирус старался найти в тексте книг такие 
последовательности символов, чтобы можно было составить цепочку из максимально возможного количества реплик, и при этом эта цепочка в тексте 
должна быть непрерывной. Полученную цепочку из реплик вирус из текста удалял.

Пример
Вирус: abcd
Строка текста: acdbababcdabcdbca
Вирус отдаст предпочтение зелёной подстроке, состоящей из 2-х реплик, и вырежет именно её.
Останется текст: acdbabbca
Вирус не возвращается «доедать текст».

Формат ввода
(в условии не указан явно, но по смыслу):
Две строки:
1-я — строка-вирус (длина ≤ 10)
2-я — текст книги (длина ≤ 500)

Формат вывода
Результирующая строка после удаления цепочки из максимального числа подряд идущих реплик вируса.
