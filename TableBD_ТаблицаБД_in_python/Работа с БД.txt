                #10. Поиск по минимальной зарплате

def find_employees_by_min_salary(min_salary):
  cursor.execute(
      "SELECT name, salary, position FROM employees WHERE salary > ? ORDER BY salary DESC",
      (min_salary,)
  )
  return cursor.fetchall()

high_earners = find_employees_by_min_salary(125000)
for emp in high_earners:
  print(f"{emp[0]} - {emp[1]} руб. {emp[2]}")

Команды:
A. Параметризованный запрос (?):
? - заполнитель для параметра
(min_salary,) - кортеж с передаваемым значением (запятая важна для кортежа из одного элемента)
Почему это важно? Это защищает от SQL-инъекций. Значение min_salary безопасно подставляется в запрос, а не встраивается как часть строки.

B. Структура запроса:
SELECT name, salary, position - выбираем только нужные столбцы
FROM employees - из таблицы сотрудников
WHERE salary > ? - фильтр: зарплата больше заданного значения
ORDER BY salary DESC - сортировка по зарплате по убыванию (от самой высокой)


     11.    Общая зарплата сотрудников. Средняя зарплата сотрудников. Подсчет количества сотрудников


cursor.execute(
    "SELECT SUM(salary), AVG(salary), COUNT(*) FROM employees"
)
stats = cursor.fetchone() # stats будет кортежем с тремя элементами: (sum_salary, avg_salary, employee_count)

print(f"Общий фонд: {stats[0]:,} руб.")
print(f"Средняя зарплата: {stats[1]:0f} руб.")
print(f"Количество сотрудников: {stats[2]} чел.")

Команды:
1) SUM(salary) - сумма всех зарплат (общий фонд оплаты труда)
2) AVG(salary) - среднее арифметическое всех зарплат
3) COUNT(*) - количество записей (сотрудников) в таблице
Все три значения возвращаются одной строкой (одним кортежем).
