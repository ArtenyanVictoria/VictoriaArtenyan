# VictoriaArtenyan

Меня зовут Артенян Виктория. Я выпускница школы и изучаю Python самостоятельно.

В этом репозитории собраны мои проекты и решения задач, выполненные в процессе изучения программирования и алгоритмов. Основная цель репозитория — продемонстрировать приобретённые навыки программирования, умение решать алгоритмические задачи и интерес к компьютерным наукам.

## Проекты

### Крестики-нолики

Консольная реализация игры «Крестики-нолики».

Изученные темы:

* функции;
* условия и циклы;
* организация логики программы;
* взаимодействие с пользователем.

### Парсинг сайтов

Программы для получения и обработки данных с веб-страниц.

Изученные темы:

* HTTP-запросы;
* обработка HTML;
* автоматизация сбора данных.

### Регистрация и вход пользователя

Консольное приложение для регистрации и авторизации пользователей.

Изученные темы:

* работа с функциями;
* проверка корректности данных;
* хранение информации о пользователях.

### Работа с JSON и CSV

Программы для чтения, обработки и сохранения данных.

Изученные темы:

* работа с файлами;
* сериализация данных;
* обработка структурированной информации.
